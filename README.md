# 🚀 AI Vision & Modbus 통신 통합 제어 프로젝트

이 레포지토리는 아두이노(Arduino) 실습 키트와 C# WinForms를 활용하여 **모드버스(Modbus) 통신 기반의 하드웨어 제어** 및 **Cognex 비전 센서 연동**을 구현한 실무 중심의 연습 프로젝트입니다.

## 🛠️ 개발 환경 및 기술 스택
* **Hardware:** Arduino Uno 실습 키트, 엑추에이터(서보모터, RGB LED, LCD, 부저), 각종 센서, Cognex 머신 비전 센서
* **Software/Language:** C# (Windows Forms), C++ (Arduino Framework)
* **Communication:** Modbus RTU (Serial)

---

## ✨ 주요 구현 기능

### 1. Modbus 통신 기반 C# WinForms UI 구축
* 아두이노(Slave)와 PC(Master) 간의 안정적인 모드버스 시리얼 연결 구현
* 통신 연결 상태 및 시스템 동작 상태를 UI 상에 직관적으로 표시

### 2. 다중 센서 실시간 모니터링 시스템
* 하드웨어 모듈에서 수집된 데이터를 모드버스 레지스터를 통해 C# UI로 실시간 수신 및 시각화
* **적용 센서 목록:** 초음파(거리), 조도(빛), 온도, 습도, 가변저항(다이얼), 홀(자성), 마이크(소리) 센서

### 3. 양방향 하드웨어 원격 제어 (UI ➡️ Arduino)
* **RGB LED 아날로그 제어:** UI 내 트랙바(TrackBar)를 조작하여 RGB 모듈의 색상 및 밝기를 실시간으로 조율
* **보드 LCD 텍스트 출력:** UI 텍스트박스에 입력한 문자열을 전송하여 하드웨어 LCD 디스플레이에 즉시 출력
* **서보모터 각도 제어:** UI를 통해 입력된 목표 각도(0~180도) 값에 맞춰 서보모터 정밀 구동

### 4. 데이터 기반 조건부 알림 기능
* 수신되는 센서 데이터의 변화량을 감지하여 설정된 조건 값 도달 시 하드웨어 부저(경고음) 자동 작동

### 5. Cognex 머신 비전 센서 데이터 연동
* 비전 검사 모니터링을 위한 전용 C# UI (`Vision_UI`) 별도 구축
* Cognex 비전 센서의 촬영(Trigger) 제어 및 검출/판독 결과(X, Y, Angle, QR 등) 데이터를 UI에 실시간 연동 및 판정(OK/NG) 표시

## 📁 디렉토리 구조 및 주요 파일
* **`WinForm_UI/`**
  * 모드버스 통신, 센서 데이터 모니터링 및 하드웨어 제어를 담당하는 C# 윈폼 메인 프로젝트
* **`Vision_UI/`**
  * Cognex 비전 센서 연동 및 결과 확인을 위한 C# 윈폼 프로젝트
* **`src/` 및 `*.cpp` 파일들**
  * `modbus.cpp`, `DHT11.cpp`, `echo.cpp`, `main.cpp` 등 아두이노 모듈 제어 및 모드버스 슬레이브 동작을 처리하는 펌웨어 소스 코드
